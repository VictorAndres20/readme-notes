# Prompt Notes

------------------------------------------------------------------------------------------------------------------------------------------

## Best practices

Be detailed and specific 
- Separate context and instructions using delimiters.

  ```text
  Take a look at this error delimieted by """ -
  """ NUll pointer exception on ine..."""
  ```

- Use positive prompts.

  ```text
  I am interested in ...
  I like to get...
  The best way to do it is...
  ```

- Provide steps for the desire outcome.

  ```text
  ## Steps to follow:
  1. Investigate patterns in code base.
  2. Investigate relationships.
  3. Create new entities.
  4. Update repository functions
  ...
  ```

- Break large tasks into smaller sections.

- Give examples.
